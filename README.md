# Видео-сетка (drag & drop, iOS-стиль)

Веб-приложение: 4 шаблона раскладки экрана, до 4 ячеек с видео одновременно,
16 видео в нижней выезжающей шторке — перетаскиваешь превью в нужную ячейку.

## 1. Добавь свои видео

Положи 16 файлов **прямо в корень репозитория** (рядом с index.html), с именами:

```
video1.mp4
video2.mp4
...
video16.mp4
```

Если видео меньше 16 — ничего страшного, просто у оставшихся ячеек-превью
в шторке не будет картинки (будет виден номер вместо превью), их можно
не трогать. Если хочешь другие имена файлов — поменяй в `script.js` строку:

```js
const videos = Array.from({ length: VIDEO_COUNT }, (_, i) => `video${i + 1}.mp4`);
```

на явный список своих имён, например:

```js
const videos = ["clip_a.mp4", "clip_b.mp4", ...];
```

и поставь `VIDEO_COUNT` равным длине этого списка.

## 2. Как пользоваться

- **Сверху** — 4 кнопки-шаблона (2×2 / два ряда / большая+2 маленьких / два столбца).
  Тап переключает раскладку экрана, уже вставленные видео не пропадают.
- **Снизу** — узкая полоска-ручка. Свайп вверх (или тап) — открывает список
  из 16 превью. Свайп вниз (или повторный тап) — закрывает.
- Превью в списке зажимаешь пальцем и тащишь в нужную ячейку — видео начнёт
  играть там сразу, зациклено, без звука (так требует iOS для автозапуска).
- На каждой заполненной ячейке — крестик (убрать видео) и значок звука
  (включить/выключить звук у конкретного видео).

## 3. Публикация на GitHub Pages

Так же, как раньше: закинуть все файлы в корень репозитория (через
Add file → Upload files или через встроенный редактор GitHub), затем
Settings → Pages → Source: branch `main`, папка `/ (root)`.

## 4. Установка на iPhone

Открой ссылку на сайт в Safari → «Поделиться» → **На экран «Домой»**.

## Структура проекта

```
index.html      — разметка: шаблоны, сетка ячеек, шторка
style.css       — белый iOS-стиль, все 4 раскладки через CSS Grid
script.js       — переключение шаблонов, генерация превью, drag & drop
manifest.json   — манифест PWA
sw.js           — офлайн-кэш статики (видео не кэшируются)
icon-192.png / icon-512.png — иконки для домашнего экрана
video1.mp4 ... video16.mp4  — твои видео (нужно добавить)
```
