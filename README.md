# Видео-плеер (Liquid Glass)

Простое веб-приложение: слева — 4 стеклянные кнопки выбора видео, справа —
плеер в закруглённом прямоугольнике. Работает как обычная веб-страница и
как PWA (можно добавить на домашний экран iPhone).

## 1. Добавь свои видео

Положи 4 файла в папку `videos/` строго с такими именами:

```
videos/video1.mp4
videos/video2.mp4
videos/video3.mp4
videos/video4.mp4
```

Если хочешь другие имена или расширения — поменяй список `videos` в начале
файла `script.js`:

```js
const videos = [
  "videos/video1.mp4",
  "videos/video2.mp4",
  "videos/video3.mp4",
  "videos/video4.mp4"
];
```

Подписи на кнопках меняются в `index.html` — найди `<span class="btn-label">`
и впиши свой текст.

> Совет: для веба лучше использовать H.264 (.mp4), а не HEVC — так видео
> точно проиграется в любом браузере. Если файлы большие, GitHub Pages
> ограничивает размер репозитория (обычно комфортно до ~100 МБ на файл,
> но чем меньше — тем быстрее грузится страница на мобильном интернете).

## 2. Выложи на GitHub

```bash
git init
git add .
git commit -m "Video glass app"
git branch -M main
git remote add origin https://github.com/<твой-юзернейм>/<репозиторий>.git
git push -u origin main
```

Дальше в настройках репозитория: **Settings → Pages → Source: `main` branch,
`/ (root)`**. Через минуту-две сайт будет доступен по адресу вида:

```
https://<твой-юзернейм>.github.io/<репозиторий>/
```

## 3. Установи на iPhone как приложение

1. Открой ссылку на сайт в **Safari** на iPhone.
2. Нажми кнопку "Поделиться" (квадрат со стрелкой вверх).
3. Выбери **«На экран «Домой»»**.
4. Иконка появится на домашнем экране и будет открываться в полноэкранном
   режиме, без адресной строки браузера — как настоящее приложение.

## Структура проекта

```
index.html      — разметка (кнопки + плеер)
style.css       — стиль Liquid Glass для кнопок, layout
script.js       — переключение видео по клику
manifest.json   — манифест PWA (иконка, имя, режим отображения)
sw.js           — service worker (офлайн-кэш статики)
icons/          — иконки для домашнего экрана (замени на свои при желании)
videos/         — сюда кладёшь свои 4 видео
```

## Заметки по дизайну

- Стеклянный эффект (`backdrop-filter: blur()`) применён только к кнопкам
  слева — это навигационный слой. Сам плеер справа — обычный закруглённый
  прямоугольник без стекла, поверх видео стекло не накладывается
  (по гайдлайну Apple Liquid Glass это не рекомендуется для медиаконтента).
- В портретной ориентации макет автоматически перестраивается: видео сверху,
  кнопки — горизонтальной строкой снизу (так удобнее держать телефон вертикально).
- Если хочешь заменить иконки для домашнего экрана — просто перезапиши
  `icons/icon-192.png` и `icons/icon-512.png` своими PNG того же размера.
