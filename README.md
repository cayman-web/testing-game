# Видео-сетка (drag & drop, iOS-стиль)

Веб-приложение: гибкая раскладка от 1 до 4 видео одновременно, 16 видео
в нижней выезжающей шторке — перетаскиваешь превью в нужное место экрана.

## 1. Добавь свои видео

Положи 16 файлов **прямо в корень репозитория** (рядом с index.html), с именами:

```
video1.mp4
video2.mp4
...
video16.mp4
```

Если видео меньше 16 — не страшно, у оставшихся превью в шторке просто
не будет картинки (вместо неё — номер). Если хочешь другие имена —
поменяй в `script.js`:

```js
const videos = Array.from({ length: VIDEO_COUNT }, (_, i) => `video${i + 1}.mp4`);
```

на явный список, например `["clip_a.mp4", "clip_b.mp4", ...]`, и поставь
`VIDEO_COUNT` равным длине списка.

## 2. Как пользоваться

- **Внизу** — узкая полоска-ручка. Свайп вверх (или тап) открывает список
  из 16 превью (первый кадр видео, сгенерированный автоматически). Свайп
  вниз (или повторный тап) закрывает.
- Пока экран пустой — одна большая ячейка на весь экран. Перетащи туда
  превью — видео начнёт играть (зациклено, без звука — так требует iOS
  для автозапуска).
- Хочешь добавить второе видео — перетащи ещё одно превью **поверх уже
  играющего видео**. Появится вопрос: разместить новое видео сверху/снизу
  или слева/справа. Выбираешь — экран делится пополам, старое видео
  «поджимается», новое встаёт рядом.
- Так же можно продолжать делить дальше — вплоть до 4 видео одновременно.
  Когда все 4 ячейки заняты, перетаскивание нового превью на любую из них
  просто заменяет видео в этой ячейке (делить больше некуда).
- Крестик на ячейке — убрать видео (ячейка станет пустой, форма экрана
  при этом не меняется, в неё можно перетащить другое видео).
- Значок динамика на ячейке — включить/выключить звук именно у этого видео.
- **Двойной тап по видео** — открывает его на весь экран (с нативными
  элементами управления воспроизведением). Крестик в углу — закрыть.

## 3. Публикация на GitHub Pages

Всё так же: файлы в корень репозитория (Add file → Upload files или
через встроенный редактор), затем Settings → Pages → Source: branch
`main`, папка `/ (root)`.

## 4. Установка на iPhone

Открой ссылку на сайт в Safari → «Поделиться» → **На экран «Домой»**.

## Структура проекта

```
index.html      — разметка: сетка ячеек (строится JS-ом), шторка, попапы
style.css       — белый iOS-стиль, гибкие flex-раскладки
script.js       — дерево разбиений экрана, превью, drag & drop, фуллскрин
manifest.json   — манифест PWA
sw.js           — офлайн-кэш статики (видео не кэшируются)
icon-192.png / icon-512.png — иконки для домашнего экрана
video1.mp4 ... video16.mp4  — твои видео (нужно добавить)
```
